이 페이지는 Appium과 WebDriver를 통한 iOS 및 Safari 자동화 기능을 문서화합니다. iOS 자동화는 iOS simulator와 USB로 연결된 실제 device에서 실제 Mobile Safari 동작을 테스트할 수 있게 합니다.

## 개요

`agent-browser`의 iOS 자동화는 desktop browser 자동화와 다른 기술 stack을 사용합니다. Playwright나 CDP 대신 Appium을 통한 WebDriver protocol을 사용해 iOS Simulator 또는 실제 device에서 실행되는 Safari를 제어합니다 [docs/src/app/ios/page.mdx:1-5](). 이 시스템은 `snapshot`, `fill` 같은 핵심 command의 동등성을 유지하면서 `tap`, `swipe` 같은 특화된 mobile interaction을 지원합니다 [docs/src/app/ios/page.mdx:41-62]().

### 아키텍처 비교

**Desktop:** CLI → Daemon → Playwright → Chrome (직접 CDP).

**iOS:** CLI → Daemon → `AppiumManager` → `WebDriverClient` → Appium → XCUITest → Simulator → Safari.

## 설정 요구 사항

iOS 자동화에는 Xcode와 Appium ecosystem이 설치된 macOS가 필요합니다 [docs/src/app/ios/page.mdx:6-10]().

### 설치 단계

**1. Appium과 XCUITest driver 설치:**

```bash
npm install -g appium
appium driver install xcuitest
```

**2. Xcode 설치 확인:**

```bash
xcode-select --install  # If needed
xcrun simctl list devices available  # List available simulators
```

**3. agent-browser로 테스트:**

```bash
agent-browser device list
```

Sources: [docs/src/app/ios/page.mdx:12-20](), [docs/src/app/ios/page.mdx:27-39]()

## Device 선택 및 관리

### 사용 가능한 Device 나열

`device list` command는 사용 가능한 iOS simulator를 조회합니다. macOS에서는 일반적으로 `xcrun simctl`과 interface하여 UDID와 device 이름을 나열합니다 [docs/src/app/ios/page.mdx:22-39]().

```bash
agent-browser device list
```

### Device 지정

`--device` flag 또는 `AGENT_BROWSER_IOS_DEVICE` 환경 변수를 사용합니다. `AppiumManager`는 이를 사용해 필요한 `XCUITest` capabilities를 구성합니다 [cli/src/native/webdriver/appium.rs:43-70]().

| 변수 | 설명 |
|----------|-------------|
| `AGENT_BROWSER_PROVIDER` | iOS mode를 활성화하려면 `ios`로 설정 |
| `AGENT_BROWSER_IOS_DEVICE` | Device 이름(예: "iPhone 16 Pro") |
| `AGENT_BROWSER_IOS_UDID` | Device UDID(name의 대안) |

Sources: [docs/src/app/ios/page.mdx:80-103](), [cli/src/native/webdriver/appium.rs:43-70]()

### 세션 초기화 흐름

1. **Connection Check:** `AppiumManager::connect_or_launch`는 Appium server가 이미 `APPIUM_DEFAULT_PORT`(4723)에서 실행 중인지 확인합니다 [cli/src/native/webdriver/appium.rs:17-28]().
2. **Launch:** 실행 중이 아니면 `launch_appium`이 `npx` 또는 `--relaxed-security`를 포함한 직접 command로 `appium` process를 spawn합니다 [cli/src/native/webdriver/appium.rs:166-190]().
3. **Capabilities:** `build_ios_capabilities`는 `platformName: "iOS"`, `appium:automationName: "XCUITest"`, `browserName: "Safari"`를 포함하는 JSON payload를 구성합니다 [cli/src/native/webdriver/appium.rs:43-70]().
4. **Session Creation:** `create_ios_session`은 `WebDriverClient`를 호출해 `/session`으로 POST하고 결과 `sessionId`를 저장합니다 [cli/src/native/webdriver/appium.rs:72-83]().

## Mobile 전용 Command

### tap과 swipe

iOS 자동화는 WebDriver "actions"로 mapping되는 native touch gesture를 도입합니다.

*   **Tap:** `AppiumManager::tap`은 `pointerMove`, `pointerDown`, 100ms `pause`, `pointerUp`을 포함하는 `touch` type의 `pointer` action sequence를 구성합니다 [cli/src/native/webdriver/appium.rs:85-105]().
*   **Swipe:** `AppiumManager::swipe`는 제공된 `duration_ms`가 있는 `pointerMove`를 사용해 지정된 duration 동안 시작 좌표에서 끝 좌표로 이동합니다 [cli/src/native/webdriver/appium.rs:107-134]().

```bash
# Swipe gestures
agent-browser -p ios swipe up
agent-browser -p ios swipe down 500

# Tap (alias for click, semantically clearer for touch)
agent-browser -p ios tap @e1
```

Sources: [docs/src/app/ios/page.mdx:64-78](), [cli/src/native/webdriver/appium.rs:85-134]()

## 실제 Device 지원

USB로 연결된 실제 iOS device가 지원됩니다. 이를 위해서는 `WebDriverAgent`(WDA)가 device에 서명되어 설치되어 있어야 합니다 [docs/src/app/ios/page.mdx:115-118]().

### 실제 Device 설정
1. **UDID:** `xcrun xctrace list devices`를 사용해 device를 식별합니다 [docs/src/app/ios/page.mdx:120-128]().
2. **Signing:** Xcode에서 `WebDriverAgent.xcodeproj`를 열고 `WebDriverAgentRunner` target에 development team을 설정합니다 [docs/src/app/ios/page.mdx:132-147]().
3. **Trust:** device에서 **Settings → General → VPN & Device Management**로 이동해 developer certificate를 신뢰하도록 설정합니다 [docs/src/app/ios/page.mdx:164]().

Sources: [docs/src/app/ios/page.mdx:115-165]()

## CDP 대비 제한 사항

iOS 자동화는 실제 mobile testing을 제공하지만, WebDriver의 abstraction layer 때문에 CDP 기반 desktop stack에서 사용할 수 있는 일부 기능은 없습니다 [docs/src/app/ios/page.mdx:172-185]().

| 기능 | Desktop (CDP) | iOS (WebDriver/Appium) |
|---------|---------------|-------------------------|
| **Browser** | Chrome, Lightpanda | Safari 전용 |
| **Tabs** | 지원됨 | 단일 Tab만 가능 |
| **PDF Export** | 지원됨 | 지원되지 않음 |
| **Screencast** | 지원됨 | 지원되지 않음 |
| **Gestures** | Simulated | Native (Tap/Swipe) |

Sources: [docs/src/app/ios/page.mdx:172-185]()
