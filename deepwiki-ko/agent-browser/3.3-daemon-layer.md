## Native Rust Daemon кө¬нҳ„

native daemonмқҖ н•өмӢ¬ background processмһ…лӢҲлӢӨ. binaryк°Җ internal daemon flagлЎң мӢӨн–үлҗ  л•Ң triggerлҗҳл©°, мқјл°ҳм ҒмңјлЎң CLIмқҳ `ensure_daemon`м—җ мқҳн•ҙ мӢңмһ‘лҗ©лӢҲлӢӨ.

### Lifecycle Management
daemon lifecycleмқҖ `run_daemon` [cli/src/native/daemon.rs:19-150]()м—җм„ң кҙҖлҰ¬лҗ©лӢҲлӢӨ. лӢӨмқҢ лӢЁкі„лҘј мҲҳн–үн•©лӢҲлӢӨ.
1. **Log Redirection**: Unixм—җм„ң `AGENT_BROWSER_DEBUG`к°Җ м„Өм •лҗҳм–ҙ мһҲмңјл©ҙ stderrк°Җ `{session}.log`лЎң redirectлҗ©лӢҲлӢӨ [cli/src/native/daemon.rs:29-43](). к·ёл Үм§Җ м•Ҡмңјл©ҙ parent CLIк°Җ pipeлҘј лӢ«м•ҳмқ„ л•Ң crashлҘј л°©м§Җн•ҳкё° мң„н•ҙ `/dev/null`лЎң redirectлҗ©лӢҲлӢӨ [cli/src/native/daemon.rs:49-60]().
2. **PID & Versioning**: process IDлҘј `{session}.pid`м—җ м“°кі  [cli/src/native/daemon.rs:62-63](), package versionмқ„ `{session}.version`м—җ м”ҒлӢҲлӢӨ [cli/src/native/daemon.rs:65-66]().
3. **State Cleanup**: `AGENT_BROWSER_STATE_EXPIRE_DAYS`к°Җ м„Өм •лҗҳм–ҙ мһҲмңјл©ҙ м„ нғқм ҒмңјлЎң `state_clean`мқ„ мӢӨн–үн•©лӢҲлӢӨ [cli/src/native/daemon.rs:88-94]().
4. **Stream Server**: WebSocket previewмҷҖ dashboard dataлҘј мІҳлҰ¬н•ҳкё° мң„н•ҙ `StreamServer::start_without_client`лҘј нҶөн•ҙ `StreamServer`лҘј мӢңмһ‘н•©лӢҲлӢӨ [cli/src/native/daemon.rs:102-113]().
5. **Socket Server**: л“Өм–ҙмҳӨлҠ” IPC commandлҘј мІҳлҰ¬н•ҳкё° мң„н•ҙ main `run_socket_server` loopм—җ м§„мһ…н•©лӢҲлӢӨ [cli/src/native/daemon.rs:122-129]().
6. **Idle Timeout**: `AGENT_BROWSER_IDLE_TIMEOUT_MS`к°Җ м„Өм •лҗҳм–ҙ мһҲмңјл©ҙ daemonмқҖ inactivityлҘј monitoringн•ҳкі  `mpsc` reset channelмқ„ нҶөн•ҙ мһҗлҸҷ мў…лЈҢлҗ©лӢҲлӢӨ [cli/src/native/daemon.rs:117-120](), [cli/src/native/daemon.rs:176-188]().

### StreamServerмҷҖ WebSocket Protocol
`StreamServer` [cli/src/native/stream/mod.rs:45-64]()лҠ” dashboardлҘј мң„н•ң real-time observation interfaceлҘј м ңкіөн•©лӢҲлӢӨ.

- **Dual Protocol Support**: serverлҠ” `handle_connection`м—җм„ң incoming connectionмқ„ WebSocket(streamingмҡ©)кіј HTTP(dashboard asset/APIмҡ©) мӮ¬мқҙм—җ dispatchн•©лӢҲлӢӨ [cli/src/native/stream/websocket.rs:99-147]().

- **Status Broadcasting**: нҳ„мһ¬ `engine`, `connected` state, `recording` statusлҘј нҸ¬н•Ён•ҳлҠ” `status` JSON messageлҘј мЈјкё°м ҒмңјлЎң broadcastн•©лӢҲлӢӨ [cli/src/native/stream/cdp_loop.rs:88-97]().
- **Viewport Coordination**: screencast frameмқҙ browserмқҳ нҳ„мһ¬ stateмҷҖ мқјм№ҳн•ҳлҸ„лЎқ viewport dimensionмқ„ м¶”м Ғн•©лӢҲлӢӨ [cli/src/native/stream/mod.rs:108-121]().

## IPC Mechanisms

daemonмқҖ CLI clientмҷҖмқҳ нҶөмӢ мқ„ мң„н•ҙ л‘җ к°Җм§Җ IPC transportлҘј м§Җмӣҗн•©лӢҲлӢӨ.

### Unix Domain Sockets
Unix кі„м—ҙ systemм—җм„ң daemonмқҖ `UnixListener`лҘј нҶөн•ҙ listenн•©лӢҲлӢӨ [cli/src/native/daemon.rs:160-163](). socket directoryлҠ” environment variable лҳҗлҠ” system defaultлҘј кё°л°ҳмңјлЎң resolveлҗҳм–ҙ session м „л°ҳм—җм„ң мқјкҙҖлҗң accessлҘј ліҙмһҘн•©лӢҲлӢӨ.

### TCP Sockets (Windows)
Windowsм—җм„ң daemonмқҖ `127.0.0.1`мқҳ TCPлҘј мӮ¬мҡ©н•©лӢҲлӢӨ.
- **Port Resolution**: clientлҠ” daemonмқҙ мһ‘м„ұн•ң `{session}.port` fileмқ„ мқҪм–ҙ portлҘј resolveн•©лӢҲлӢӨ.
- **Cleanup**: daemonмқҖ startup мӢң stale port fileкіј socketмқҙ м ңкұ°лҗҳлҸ„лЎқ ліҙмһҘн•©лӢҲлӢӨ [cli/src/native/daemon.rs:77-86]().

**м¶ңмІҳ:** [cli/src/native/daemon.rs:68-86](), [cli/src/native/daemon.rs:160-163]()

## Process Coordination

### `ensure_daemon` Flow
CLI clientлҠ” commandлҘј ліҙлӮҙкё° м „м—җ daemonмқҙ active мғҒнғңмқём§Җ нҷ•мқён•©лӢҲлӢӨ.

```mermaid
sequenceDiagram
    participant CLI as "CLI Client (main.rs)"
    participant CONN as "connection.rs"
    participant FS as "Filesystem"
    participant D as "Daemon Process (daemon.rs)"

    CLI->>CONN: "ensure_daemon(session, opts)"
    CONN->>CONN: "Check is_pid_alive(pid)"
    alt Already Running
        CONN-->>CLI: "DaemonResult { already_running: true }"
    else Not Running
        CONN->>FS: "cleanup_stale_files(session)"
        CONN->>FS: "Command::spawn(daemon_args)"
        loop Poll for socket/port
            CONN->>CONN: "attempt connection"
            alt Connected
                CONN-->>CLI: "DaemonResult { already_running: false }"
            else Timeout (5s)
                CONN-->>CLI: "Error"
            end
        end
    end
```

### Chrome Process Management
daemonмқҖ `ChromeProcess` [cli/src/native/cdp/chrome.rs:8-15]()лҘј нҶөн•ҙ browser lifecycleмқ„ м§Ғм ‘ кҙҖлҰ¬н•©лӢҲлӢӨ.
- **Process Group Termination**: Unixм—җм„ң `ChromeProcess::kill`мқҖ лӘЁл“  helper process(GPU, renderer)к°Җ мў…лЈҢлҗҳлҸ„лЎқ process group ID(`pgid`)лҘј мӮ¬мҡ©н•©лӢҲлӢӨ [cli/src/native/cdp/chrome.rs:18-31]().
- **Wait or Kill**: daemonмқҖ force-killing м „м—җ cookieлҘј flushн•  мҲҳ мһҲлҸ„лЎқ timeout лҸҷм•Ҳ Chromeмқҙ gracefulн•ҳкІҢ exitн•ҳлҸ„лЎқ мӢңлҸ„н•©лӢҲлӢӨ [cli/src/native/cdp/chrome.rs:47-60]().
