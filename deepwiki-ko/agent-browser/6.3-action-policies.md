## лӘ©м Ғкіј Threat Model

action policyлҠ” AI agentк°Җ harmful operationмқ„ мӢңлҸ„н•  мҲҳ мһҲлҠ” scenarioлҘј лӢӨлЈ№лӢҲлӢӨ.

*   **Code execution**: CDP `Runtime.evaluate` domainмқ„ нҶөн•ҙ malicious JavaScriptлҘј мӢӨн–үн•ҳлҠ” `eval` command.
*   **Data exfiltration**: sensitive fileмқ„ downloadн•ҳкұ°лӮҳ session cookieлҘј м „мҶЎн•ҳлҠ” н–үмң„.
*   **Destructive mutations**: storage clear, data deletion, unauthorized form submission.
*   **Resource abuse**: кіјлҸ„н•ң tab м—ҙкё° лҳҗлҠ” unbounded network request мҲҳн–ү.

policy engineмқҖ лӘЁл“  browser commandлҘј risk levelлі„лЎң categorizationн•ҳкі , execution м „м—җ allow/deny/confirm ruleмқ„ м Ғмҡ©н•©лӢҲлӢӨ. мқҙлҘј нҶөн•ҙ LLMмқҙ valid command payloadлҘј мғқм„ұн•ҳлҚ”лқјлҸ„ лӘ…мӢңм Ғ approval м—Ҷмқҙ daemonмқҙ dangerous commandлҘј мӢӨн–үн•ҳм§Җ лӘ»н•ҳкІҢ н•©лӢҲлӢӨ.

## Policy File Format

policyлҠ” `ActionPolicy` structлЎң н‘ңнҳ„лҗҳлҠ” JSON fileм—җ м •мқҳлҗ©лӢҲлӢӨ. мқҙ formatмқҖ explicit listмҷҖ fallback defaultлҘј м§Җмӣҗн•©лӢҲлӢӨ.

```json
{
  "default": "deny",
  "allow": ["navigate", "snapshot", "click", "scroll", "wait", "get"],
  "deny": ["eval", "download", "storage_clear", "cookies_clear"],
  "confirm": ["submit"]
}
```

### Fields

| Field | Type | Description |
|-------|------|-------------|
| `default` | `Option<String>` | fallback policy: `"allow"`, `"deny"`, лҳҗлҠ” `"confirm"`. |
| `allow` | `Option<Vec<String>>` | мҰүмӢң мӢӨн–үлҗҳлҠ” action category. |
| `deny` | `Option<Vec<String>>` | мҰүмӢң errorлҘј л°ҳнҷҳн•ҳлҠ” action category. |
| `confirm` | `Option<Vec<String>>` | manual approvalмқҙ н•„мҡ”н•ң action category. |

`default` fieldлҠ” лӘ…мӢңм ҒмңјлЎң listedлҗҳм§Җ м•ҠмқҖ лӘЁл“  actionм—җ м Ғмҡ©лҗ©лӢҲлӢӨ. `default: "deny"`мҷҖ explicit `allow` listлҘј м„Өм •н•ҳл©ҙ strict allowlist-only policyк°Җ л§Ңл“Өм–ҙм§‘лӢҲлӢӨ.

## Enforcement Modes

`PolicyResult` enumмқҖ мЈјм–ҙм§„ action checkм—җ лҢҖн•ң м„ё к°Җм§Җ outcomeмқ„ м •мқҳн•©лӢҲлӢӨ.

### PolicyResult::Allow
actionмқҖ user interaction м—Ҷмқҙ мҰүмӢң мӢӨн–үлҗ©лӢҲлӢӨ. policyк°Җ м ңкіөлҗҳм§Җ м•Ҡм•ҳмқ„ л•Ңмқҳ default behaviorмһ…лӢҲлӢӨ.

### PolicyResult::Deny
actionмқҖ daemon levelм—җм„ң blockedлҗ©лӢҲлӢӨ. daemonмқҖ `"success": false`мҷҖ policy violationмқ„ лӮҳнғҖлӮҙлҠ” error messageк°Җ нҸ¬н•Ёлҗң JSON responseлҘј л°ҳнҷҳн•©лӢҲлӢӨ.

### PolicyResult::RequiresConfirmation
actionмқҖ лӘ…мӢңм Ғ approvalмқҙ н•„мҡ”н•©лӢҲлӢӨ. daemonмқҖ commandлҘј `PendingConfirmation` structм—җ м ҖмһҘн•ҳкі  `confirmation_required` payloadлҘј л°ҳнҷҳн•©лӢҲлӢӨ.

## Implementation Details

### Policy Decision Logic
`cli/src/native/policy.rs`мқҳ `check` methodлҠ” explicit listлҘј нҷ•мқён•ҳм—¬ actionмқҳ fateлҘј кІ°м •н•©лӢҲлӢӨ. `Deny`лҠ” н•ӯмғҒ `Confirm`ліҙлӢӨ мҡ°м„ н•ҳкі , `Confirm`мқҖ `Allow`ліҙлӢӨ мҡ°м„ н•©лӢҲлӢӨ.

```rust
pub fn check(&self, action: &str) -> PolicyResult {
    if let Some(deny) = &self.deny {
        if deny.iter().any(|a| a == action) {
            return PolicyResult::Deny(format!("Action '{}' is denied by policy", action));
        }
    }

    if let Some(confirm) = &self.confirm {
        if confirm.iter().any(|a| a == action) {
            return PolicyResult::RequiresConfirmation;
        }
    }
    // ... logic for allow and default
}
```

## Security Considerations

1.  **Strict Defaulting**: `default: "deny"`лҘј м„Өм •н•ҳл©ҙ operatorлҠ” мөңмҶҢн•ңмқҳ safe action(`snapshot`, `get` л“ұ)л§Ң agentк°Җ мӮ¬мҡ©н•  мҲҳ мһҲлҸ„лЎқ ліҙмһҘн•  мҲҳ мһҲмҠөлӢҲлӢӨ. [cli/src/native/policy.rs:99-110]()
2.  **Precedence**: engineмқҖ actionмқҙ мӢӨмҲҳлЎң м–‘мӘҪм—җ нҸ¬н•Ёлҗҳм—Ҳмқ„ л•Ң accidental exposureлҘј л°©м§Җн•ҳкё° мң„н•ҙ `deny` listлҘј `allow` listліҙлӢӨ мҡ°м„ н•ҳлҸ„лЎқ hardcodedлҗҳм–ҙ мһҲмҠөлӢҲлӢӨ. [cli/src/native/policy.rs:164-168]()
3.  **Default Deny Fallback**: `allow` listк°Җ м ңкіөлҗҳм—Ҳм§Җл§Ң actionмқҙ к·ё м•Ҳм—җ м—Ҷмңјл©ҙ, `default`к°Җ лӘ…мӢңм ҒмңјлЎң `allow`лЎң м„Өм •лҗҳм§Җ м•ҠмқҖ н•ң мӢңмҠӨн…ңмқҖ defaultлЎң `Deny`н•©лӢҲлӢӨ. [cli/src/native/policy.rs:97-110]()
4.  **Confirmation Timeout**: pending confirmationмқҖ resource exhaustionкіј hanging processлҘј л°©м§Җн•ҳкё° мң„н•ҙ 60мҙҲ нӣ„ мһҗлҸҷмңјлЎң denyлҗ©лӢҲлӢӨ. [docs/src/app/security/page.mdx:22-23]()
