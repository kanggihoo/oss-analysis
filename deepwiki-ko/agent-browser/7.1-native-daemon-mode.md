## 목적과 범위

native daemon mode(`run_daemon`을 통해 호출)는 기존 Node.js 기반 browser automation을 대체하는 고성능, low-overhead 방식입니다. 완전히 Rust로 구현되었으며, Chrome DevTools Protocol(CDP)을 통해 browser와 직접 통신합니다. 이 mode는 Node.js와 Playwright runtime의 필요성을 제거하여 memory consumption을 약 140MB에서 약 7MB RAM으로 줄입니다 [docs/src/app/native-mode/page.mdx:3-5](). 현재 `agent-browser`의 default architecture입니다 [docs/src/app/native-mode/page.mdx:3]().

## Daemon Lifecycle and Socket Management

daemon lifecycle은 `cli/src/native/daemon.rs`에서 관리됩니다. 시작되면 platform-specific socket directory에 session-specific lock과 communication infrastructure를 생성합니다.

### Startup Sequence
1.  **Socket Directory**: `get_daemon_socket_dir`를 통해 daemon socket directory를 찾거나 생성합니다 [cli/src/native/daemon.rs:20-23]().
2.  **Stderr Redirection**: `AGENT_BROWSER_DEBUG`가 설정되어 있으면 stderr가 `.log` file로 redirect됩니다. 그렇지 않으면 parent CLI가 pipe를 닫았을 때 crash를 방지하기 위해 Unix에서 `/dev/null`로 redirect됩니다 [cli/src/native/daemon.rs:28-60]().
3.  **PID and Versioning**: process ID가 포함된 `.pid` file과 crate version이 포함된 `.version` file을 socket directory에 씁니다 [cli/src/native/daemon.rs:62-66]().
4.  **Cleanup**: clean state를 보장하기 위해 이전 run의 stale `.sock`(Unix), `.port`(Windows), `.stream` file을 제거합니다 [cli/src/native/daemon.rs:72-87]().
5.  **Stream Server**: browser가 launch되기 전에 port에 bind할 수 있도록 `StreamServer::start_without_client`를 사용해 `StreamServer`를 initialize합니다 [cli/src/native/daemon.rs:102-109]().
6.  **Socket Server**: `run_socket_server`를 통해 main IPC listener(Unix에서는 `UnixListener`, Windows에서는 `TcpListener`)를 시작합니다 [cli/src/native/daemon.rs:122-129]().

### Idle Shutdown and Resource Cleanup
daemon은 `AGENT_BROWSER_IDLE_TIMEOUT_MS`를 통한 auto-shutdown mechanism을 지원합니다 [cli/src/native/daemon.rs:117-120](). timeout 내에 command를 받지 못하면 daemon이 exit합니다 [cli/src/native/daemon.rs:211-216](). Unix에서는 `SIGCHLD`를 listen하고 `reap_children`을 호출하여 zombie process를 reap합니다 [cli/src/native/daemon.rs:198-200](). orphaned Chrome process를 방지하기 위해 `close_notify` `Notify` instance가 "close" command 이후 graceful exit를 signal하여 destructor가 실행될 수 있게 합니다 [cli/src/native/daemon.rs:179-182]().

## Direct CDP Usage and Chrome Management

`cli/src/native/cdp/chrome.rs`의 `ChromeProcess` struct는 실제 browser binary의 lifecycle을 관리합니다.

- **Process Groups**: Unix에서는 daemon이 Chrome을 kill할 때 모든 helper process(GPU, renderer, crashpad)도 종료되도록 `pgid`를 추적합니다 [cli/src/native/cdp/chrome.rs:12-29]().
- **Graceful Shutdown**: `wait_or_kill`은 강제 종료 전에 Chrome이 cookie와 state를 `user-data-dir`에 flush할 수 있도록 timeout을 제공합니다 [cli/src/native/cdp/chrome.rs:47-60]().
- **Automatic Cleanup**: `ChromeProcess`의 `Drop` implementation은 process를 자동으로 kill하고 temporary user data directory 제거를 시도합니다 [cli/src/native/cdp/chrome.rs:63-86]().
- **Launch Options**: proxy setting, headless mode, viewport dimension을 포함한 광범위한 configuration을 `LaunchOptions`로 지원합니다 [cli/src/native/cdp/chrome.rs:89-113]().

## StreamServer and Dashboard Integration

`StreamServer`는 CDP의 `Page.screencastFrame` event를 tap하고 dashboard web UI를 serve하여 browser viewport의 real-time view를 제공합니다.

### Data Flow: Browser to Dashboard

### 핵심 구성 요소
- **Multiplexing**: `StreamServer`는 `broadcast::Sender`(`frame_tx`)를 사용해 연결된 모든 WebSocket client에 screencast frame을 동시에 보냅니다 [cli/src/native/stream/mod.rs:179]().
- **CDP Client Management**: `CdpClient`는 browser에 대한 raw WebSocket connection을 관리하고, `AtomicU64`로 command ID를 처리하며, `broadcast::Sender<CdpEvent>`를 통해 event를 broadcast합니다 [cli/src/native/cdp/client.rs:29-46]().
- **Background Event Loop**: `cdp_event_loop`는 navigation event와 screencast frame을 위해 browser를 monitoring하며, stream이 계속 흐르도록 `Page.screencastFrameAck`를 통해 frame을 자동으로 acknowledge합니다 [cli/src/native/stream/cdp_loop.rs:142-149]().
- **Provider Abstraction**: daemon은 `connect_provider`를 사용해 remote browser provider(Browserbase, Browserless 등)에 연결할 수 있으며, 이는 remote CDP WebSocket URL이 포함된 `ProviderConnection`을 반환합니다 [cli/src/native/providers.rs:26-73]().

## Performance and Resource Efficiency

native mode의 주요 장점은 minimal resource footprint이며, high-density environment 또는 constrained CI/CD runner에 적합합니다.

| Metric | Rust Daemon (Native) |
| :--- | :--- |
| **RAM Usage** | ~7MB - 15MB [docs/src/app/native-mode/page.mdx:3-5]() |
| **Binary** | Single static Rust binary |
| **Startup** | 거의 즉시(no Node.js overhead) |
| **CDP Transport** | Native WebSockets(`tokio-tungstenite`) [cli/src/native/cdp/client.rs:79-82]() |
