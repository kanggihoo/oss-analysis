## лӘ©м Ғкіј лІ”мң„

Annotation QueueлҠ” Langfuseм—җм„ң trace, observation, sessionмқ„ мҲҳлҸҷмңјлЎң reviewн•ҳкі  scoringн•ҳкё° мң„н•ң human-in-the-loop workflow systemмқ„ м ңкіөн•©лӢҲлӢӨ. мқҙ мӢңмҠӨн…ңмқ„ нҶөн•ҙ teamмқҖ manual reviewк°Җ н•„мҡ”н•ң itemмқ„ named queueлЎң кө¬м„ұн•ҳкі , нҠ№м • userлҘј assignн•ҳл©°, structured lifecycleмқ„ нҶөн•ҙ progressлҘј м¶”м Ғн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

мЈјмҡ” кё°лҠҘмқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.
- **User Assignments**: `AnnotationQueueAssignment`лҘј нҶөн•ҙ нҠ№м • team memberлЎң queue accessлҘј м ңн•ңн•©лӢҲлӢӨ. [[packages/shared/prisma/schema.prisma:566-579]]()
- **Locking Mechanism**: Active review мӨ‘ itemмқ„ мқјм • кё°к°„(мқјл°ҳм ҒмңјлЎң 5л¶„) lockн•ҳм—¬ concurrent editingмқ„ л°©м§Җн•©лӢҲлӢӨ. [[web/src/features/annotation-queues/server/annotationQueueItemsRouter.ts:32-38]]()
- **Status Tracking**: `PENDING` л°Ҹ `COMPLETED` stateлЎң progressлҘј monitorн•©лӢҲлӢӨ. [[packages/shared/prisma/schema.prisma:556-559]]()
- **Schema Standardization**: Queueм—җ `ScoreConfig` definitionмқ„ м—°кІ°н•ҳм—¬ мқјкҙҖлҗң manual evaluation criteriaлҘј ліҙмһҘн•©лӢҲлӢӨ. [[packages/shared/prisma/schema.prisma:517-517]]()
- **Integrated Feedback**: Annotation process мӨ‘ internal commentмҷҖ mentionмқ„ м§Җмӣҗн•©лӢҲлӢӨ. [[web/src/features/annotation-queues/components/processors/SessionAnnotationProcessor.tsx:60-67]]()

## AnnotationQueue лӘЁлҚё

`AnnotationQueue` tableмқҖ annotationмқҙ н•„мҡ”н•ң itemмқ„ кө¬м„ұн•ҳкё° мң„н•ң named queueлҘј м •мқҳн•©лӢҲлӢӨ.

### мҠӨнӮӨл§Ҳ

| Field | Type | Description |
|-------|------|-------------|
| `id` | string (cuid) | Primary key |
| `name` | string | Queue name(projectлі„ unique) |
| `description` | string (nullable) | Optional description |
| `scoreConfigIds` | string[] | Annotation schemaлҘј м •мқҳн•ҳлҠ” `ScoreConfig` ID л°°м—ҙ |
| `projectId` | string | `Project`м—җ лҢҖн•ң foreign key |
| `createdAt` | timestamp | Creation timestamp |
| `updatedAt` | timestamp | Last update timestamp |

### мЈјмҡ” нҠ№м„ұ

- **Unique Constraint**: `@@unique([projectId, name])`мқҖ project лӮҙм—җм„ң queue nameмқҙ uniqueн•ҳлҸ„лЎқ ліҙмһҘн•©лӢҲлӢӨ. [[packages/shared/prisma/schema.prisma:526-526]]()

## Annotation Workflow

### кө¬нҳ„кіј лҚ°мқҙн„° нқҗлҰ„

Frontend implementationмқҖ лӢӨм–‘н•ң tRPC routerлҘј нҶөн•ҙ annotation lifecycleмқ„ кҙҖлҰ¬н•ҳкё° мң„н•ҙ `AnnotationQueueItemPage`лҘј мӮ¬мҡ©н•©лӢҲлӢӨ.

**Title: Annotation Workflow Code Interaction**

```mermaid
sequenceDiagram
    participant UI as "AnnotationQueueItemPage"
    participant tRPC_AQ as "queueRouter (annotationQueues)"
    participant tRPC_AQI as "queueItemRouter (annotationQueueItems)"
    participant DB as "PostgreSQL (Prisma)"
    participant CH as "ClickHouse (Scores)"

    UI->>tRPC_AQ: "fetchAndLockNext(queueId, seenItemIds)"
    tRPC_AQ->>DB: "Update AnnotationQueueItem (set lockedAt, lockedByUserId)"
    DB-->>UI: "Return AnnotationQueueItem data"
    
    UI->>UI: "Render TraceAnnotationProcessor or SessionAnnotationProcessor"
    UI->>CH: "Create scores via AnnotationForm"
    
    UI->>tRPC_AQI: "complete(itemId)"
    tRPC_AQI->>DB: "Update status=COMPLETED, set annotatorUserId, completedAt"
    DB-->>UI: "Success"
    UI->>tRPC_AQ: "Fetch next item"
```

### лӢЁкі„лі„ н”„лЎңм„ёмҠӨ

1.  **Item Retrieval & Locking**: UIлҠ” `queueRouter`лҘј нҶөн•ҙ `fetchAndLockNext`лҘј нҳём¶ңн•©лӢҲлӢӨ. мқҙ procedureлҠ” `PENDING` itemмқ„ м°ҫкі  `lockedByUserId`лҘј м„Өм •н•©лӢҲлӢӨ. [[web/src/features/annotation-queues/components/AnnotationQueueItemPage.tsx:48-65]]()
2.  **Object Data Fetching**: `useAnnotationObjectData` hookмқҖ `objectType`мқ„ кё°л°ҳмңјлЎң underlying trace, observation, session dataлҘј resolveн•©лӢҲлӢӨ. Legacy ClickHouse pathмҷҖ `getObservationByIdFromEventsTable`мқ„ нҶөн•ң V4 Beta `events` tableмқ„ лӘЁл‘җ м§Җмӣҗн•©лӢҲлӢӨ. [[web/src/features/annotation-queues/components/shared/hooks/useAnnotationObjectData.ts:15-15]](), [[web/src/features/annotation-queues/server/annotationQueueItemsRouter.ts:140-149]]()
3.  **Annotation UI**: Typeм—җ л”°лқј systemмқҖ `TraceAnnotationProcessor` лҳҗлҠ” `SessionAnnotationProcessor`лҘј renderн•©лӢҲлӢӨ. [[web/src/features/annotation-queues/components/AnnotationQueueItemPage.tsx:212-225]]()
4.  **Scoring**: UserлҠ” `AnnotationForm`мқ„ нҶөн•ҙ feedbackмқ„ м ңкіөн•©лӢҲлӢӨ. ScoreлҠ” persistedлҗҳкі  `scoreMetadata`лҘј нҶөн•ҙ `queueId`мҷҖ м—°кІ°лҗ©лӢҲлӢӨ. [[web/src/features/annotation-queues/components/shared/AnnotationDrawerSection.tsx:38-51]]()

## Locking Mechanism

### Lock State [[packages/shared/prisma/schema.prisma:529-553]]()

| State | `lockedAt` | `lockedByUserId` | `status` | Description |
| :--- | :--- | :--- | :--- | :--- |
| **Unlocked** | `null` | `null` | `PENDING` | Itemмқ„ annotationн•  мҲҳ мһҲмқҢ |
