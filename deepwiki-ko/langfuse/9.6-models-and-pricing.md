## к°ңмҡ”

LangfuseлҠ” model definition libraryлҘј мң м§Җн•©лӢҲлӢӨ. к°Ғ definitionм—җлҠ” лӢӨмқҢмқҙ нҸ¬н•Ёлҗ©лӢҲлӢӨ.
- incoming observationмқ„ modelкіј м—°кІ°н•ҳлҠ” лҚ° мӮ¬мҡ©лҗҳлҠ” **regex match pattern**.
- countк°Җ м ңкіөлҗҳм§Җ м•Ҡм•ҳмқ„ л•Ң server-side token countingм—җ мӮ¬мҡ©лҗҳлҠ” **tokenizer configuration**.
- м„ңлЎң лӢӨлҘё usage typeкіј usage conditionм—җ лҢҖн•ң per-token costлҘј кІ°м •н•ҳлҠ” н•ҳлӮҳ мқҙмғҒмқҳ **pricing tier**.

model definitionм—җлҠ” л‘җ categoryк°Җ мһҲмҠөлӢҲлӢӨ.

| Category | DBмқҳ `projectId` | Maintainer | Editable |
|---|---|---|---|
| Langfuse-managed (built-in) | `NULL` | Langfuse | cloneл§Ң к°ҖлҠҘ |
| User-managed (custom) | Project UUID | User | edit / delete |

## Model Matching Algorithm

observationмқҙ ingestлҗҳл©ҙ(REST лҳҗлҠ” OTelмқ„ нҶөн•ҙ) pipelineмқҖ pricingмқ„ resolveн•ҳкё° мң„н•ҙ `findModel`мқ„ нҳём¶ңн•©лӢҲлӢӨ.

**Model Match Data Flow:**

```mermaid
flowchart TD
    Start(["findModel(projectId, model)"]) --> LocalCacheCheck{"LocalCache Hit?"}
    LocalCacheCheck -- "Yes" --> ReturnLocal["Return ModelWithPrices"]
    LocalCacheCheck -- "No" --> RedisCheck{"Redis Cache Enabled?"}
    RedisCheck -- "Yes" --> RedisGet["GET Redis key"]
    RedisGet -- "Hit" --> ReturnRedis["Update LocalCache & Return"]
    RedisGet -- "Miss" --> PGQuery["findModelInPostgres()"]
    RedisCheck -- "No" --> PGQuery
    PGQuery -- "Found" --> Tiers["findPricingTiersForModel()"]
    Tiers --> CacheSet["SET Redis & LocalCache"]
    CacheSet --> Return["Return ModelWithPrices"]
    PGQuery -- "Not Found" --> CacheNotFound["SET NOT_FOUND_TOKEN"]
    CacheNotFound --> ReturnNull["Return null"]
```

### Multi-Level Caching
LangfuseлҠ” high-volume ingestionмқ„ мІҳлҰ¬н•ҳкё° мң„н•ҙ model matchingм—җ two-level caching strategyлҘј кө¬нҳ„н•©лӢҲлӢӨ.
1.  **L1 Local Cache**: Redis roundtripмқ„ мӨ„мқҙкё° мң„н•ң м§§мқҖ TTL(default 10s)мқҳ in-memory `LocalCache` instance [packages/shared/src/server/ingestion/modelMatch.ts:28-43]().
2.  **L2 Redis Cache**: consistencyлҘј мң м§Җн•ҳкё° мң„н•ң container к°„ shared cache [packages/shared/src/server/ingestion/modelMatch.ts:179-206]().

### Regex Matching in Postgres
`findModelInPostgres` functionмқҖ POSIX regex match(`~` operator)лҘј мӮ¬мҡ©н•ҙ ingestлҗң model stringкіј `match_pattern`мқҙ matchingлҗҳлҠ” modelмқ„ м°ҫмҠөлӢҲлӢӨ [packages/shared/src/server/ingestion/modelMatch.ts:284-286]().

queryлҠ” project-specific modelмқ„ global modelліҙлӢӨ мҡ°м„ мӢңн•ҳл©°(`project_id ASC`), price versioningмқ„ н—Ҳмҡ©н•ҳкё° мң„н•ҙ к°ҖмһҘ мөңк·ј `start_date`лҘј м„ нғқн•©лӢҲлӢӨ [packages/shared/src/server/ingestion/modelMatch.ts:306-312]().

### Cache Invalidation
- **Locking**: `isModelMatchCacheLocked()`к°Җ trueлҘј л°ҳнҷҳн•ҳл©ҙ bulk update мӨ‘ stale dataлҘј мқҪлҠ” кІғмқ„ л°©м§Җн•ҳкё° мң„н•ҙ RedisлҘј кұҙл„ҲлңҒлӢҲлӢӨ [packages/shared/src/server/ingestion/modelMatch.ts:187-193]().
- **Negative Caching**: unknown modelм—җ лҢҖн•ҙ database hammeringмқ„ л°©м§Җн•ҳкё° мң„н•ҙ `NOT_FOUND_TOKEN`(`LANGFUSE_MODEL_MATCH_NOT_FOUND`)мқ„ мӮ¬мҡ©н•©лӢҲлӢӨ [packages/shared/src/server/ingestion/modelMatch.ts:204-206]().
- **Manual Clearing**: UI лҳҗлҠ” APIлҘј нҶөн•ҙ modelмқҙ updateлҗ  л•Ң `clearModelCacheForProject`к°Җ нҳём¶ңлҗ©лӢҲлӢӨ [web/src/server/api/routers/models.ts:18]().
