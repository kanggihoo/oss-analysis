## Git WorkflowмҷҖ Branching

м•Ҳм •м„ұмқ„ ліҙмһҘн•ҳкё° мң„н•ҙ `main` branchм—җ м§Ғм ‘ pushн•ҳлҠ” кІғмқҖ кёҲм§Җлҗ©лӢҲлӢӨ. contributorлҠ” GitHub CLI(`gh`) лҳҗлҠ” н‘ңмӨҖ Git commandлҘј нҶөн•ҙ м§„н–үлҗҳлҠ” н‘ңмӨҖ Feature Branch workflowлҘј л”°лқјм•ј н•©лӢҲлӢӨ.

### Branch Naming Conventions

| Prefix | лӘ©м Ғ | мҳҲмӢң |
| :--- | :--- | :--- |
| `feat/` | мғҲ feature лҳҗлҠ” platform | `feat/add-gemini-support` |
| `fix/` | script лҳҗлҠ” CLIмқҳ bug fix | `fix/bm25-tokenization` |
| `data/` | CSV database м—…лҚ°мқҙнҠё | `data/update-react-stack` |
| `docs/` | documentation лҳҗлҠ” wiki м—…лҚ°мқҙнҠё | `docs/testing-guide` |

### Contribution Process

1.  **Create Branch**: `git checkout -b feat/<feature-name>` [CLAUDE.md:95-95]()
2.  **Commit Changes**: conventional commitмқ„ л”°лҰ…лӢҲлӢӨ(мҳҲ: `feat: add support for Augment AI`) [CLAUDE.md:96-96]()
3.  **Push Branch**: `git push -u origin <branch>` [CLAUDE.md:97-97]()
4.  **Create Pull Request**: `gh pr create` [CLAUDE.md:98-98]()

## Local DevelopmentмҷҖ CLI Testing

`uipro-cli`лҠ” TypeScriptмҷҖ Bun runtimeмқ„ мӮ¬мҡ©н•ҙ л№Ңл“ңлҗ©лӢҲлӢӨ. н…ҢмҠӨнҠём—җлҠ” global installationмқ„ мӢңл®¬л Ҳмқҙм…ҳн•ҳкё° мң„н•ҙ local packageлҘј linkн•ҙм•ј н•©лӢҲлӢӨ.

### Environment Setup

1.  **Navigate to CLI**: `cd cli`
2.  **Install Dependencies**: `bun install` [cli/bun.lock:1-19]()
3.  **Link Package**: `bun link`(мқҙ commandлҠ” `package.json`мқҳ `bin` configurationмқ„ нҶөн•ҙ `uipro` commandлҘј globally мӮ¬мҡ©н•  мҲҳ мһҲкІҢ н•©лӢҲлӢӨ).

### Build and Test Loop

| Action | Command | File Impact |
| :--- | :--- | :--- |
| **Development** | `bun run src/index.ts <args>` | TS source м§Ғм ‘ мӢӨн–ү |
| **Build** | `bun build src/index.ts --outdir dist --target node` | `dist/index.js` мғқм„ұ |
| **Production Test** | `uipro init --ai claude` | linked `dist` output н…ҢмҠӨнҠё |

## Build and Publish Procedures

### Version Management

CLIлҠ” GitHub integrationмқ„ нҶөн•ҙ version нҷ•мқёкіј лӘ©лЎқ мЎ°нҡҢлҘј м§Җмӣҗн•©лӢҲлӢӨ.

-   **Check Versions**: `uipro versions`лҠ” `fetchReleases()`лҘј нҳём¶ңн•ҙ tagged GitHub releaseлҘј лӘЁл‘җ лӮҳм—ҙн•©лӢҲлӢӨ [cli/src/commands/versions.ts:6-10]().
-   **Update Logic**: `uipro update`лҠ” мөңмӢ  release tagлҘј к°Җм ёмҳӨкі , мөңмӢ  assetмңјлЎң local fileмқ„ overwriteн•ҳкё° мң„н•ҙ forced `initCommand`лҘј triggerн•©лӢҲлӢӨ [cli/src/commands/update.ts:12-28]().

### Release Checklist

1.  **Validate Data**: `src/ui-ux-pro-max/data/`мқҳ лӘЁл“  CSVк°Җ standard schemaлҘј л”°лҘҙлҠ”м§Җ нҷ•мқён•©лӢҲлӢӨ.
2.  **Run Sync**: `cp` commandлҘј мӢӨн–үн•ҙ `cli/assets/`лҘј м—…лҚ°мқҙнҠён•©лӢҲлӢӨ.
3.  **Verify Build**: `cli/` directoryм—җм„ң `bun run build`лҘј мӢӨн–үн•©лӢҲлӢӨ.
4.  **Tag Release**: мғҲ Git tagлҘј мғқм„ұн•©лӢҲлӢӨ(мҳҲ: `v2.5.0`).
5.  **Publish NPM**: `cli/` folderм—җм„ң `npm publish`лҘј мӢӨн–үн•©лӢҲлӢӨ.
