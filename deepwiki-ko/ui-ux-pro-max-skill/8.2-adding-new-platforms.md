| `scriptPath` | string | Yes | `search.py`лЎң к°ҖлҠ” мғҒлҢҖ кІҪлЎң | `"skills/ui-ux-pro-max/scripts/search.py"` |

| `description` | string | Yes | AI platformмқ„ мң„н•ң к°„лӢЁн•ң description | 1-2л¬ёмһҘ мҡ”м•Ҫ |
| `skillOrWorkflow` | enum | Yes | integration mode: `"Skill"` лҳҗлҠ” `"Workflow"` | auto-activateм—җлҠ” `"Skill"` |

### Skill Mode vs Workflow Mode

| Aspect | Skill Mode | Workflow Mode |
|--------|------------|---------------|
| **Activation** | UI/UX keywordм—җм„ң auto-activate | лӘ…мӢңм Ғ slash command invocation |
| **Content** | м „мІҙ knowledge base(`installType: "full"`) | Quick reference(`installType: "reference"`) |

## лӢЁкі„лі„: мғҲ Platform м¶”к°Җ

### Step 1: Platform JSON Configuration мғқм„ұ

templates directoryм—җ platform identifierлҘј filenameмңјлЎң мӮ¬мҡ©н•ҳлҠ” мғҲ fileмқ„ л§Ңл“ӯлӢҲлӢӨ.

**File:** `src/ui-ux-pro-max/templates/platforms/codeai.json`

```json
{
  "platform": "codeai",
  "displayName": "CodeAI",
  "installType": "full",
  "folderStructure": {
    "root": ".codeai",
    "skillPath": "skills/ui-ux-pro-max",
    "filename": "SKILL.md"
  },
  "scriptPath": "skills/ui-ux-pro-max/scripts/search.py",
  "frontmatter": null,
  "sections": {
    "quickReference": false
  },
  "title": "ui-ux-pro-max",
  "description": "Comprehensive design guide for web and mobile applications. Contains 67 styles, 161 color palettes, 57 font pairings, 99 UX guidelines, and 25 chart types across 16 technology stacks.",
  "skillOrWorkflow": "Skill"
}
```

**мЈјмҡ” кІ°м • мӮ¬н•ӯ:**

| Decision | Value | к·јкұ° |
|----------|-------|-----------|
| `platform` | `"codeai"` | directory detection patternкіј мқјм№ҳн•ҙм•ј н•Ё |
| `folderStructure.root` | `".codeai"` | platformмқҳ configuration directory |
| `folderStructure.skillPath` | `"skills/ui-ux-pro-max"` | platformмқҳ skill subdirectory convention |
| `folderStructure.filename` | `"SKILL.md"` | Copilot-like platformм—җлҠ” `"PROMPT.md"` мӮ¬мҡ© |
| `installType` | `"full"` | complete knowledge base нҸ¬н•Ё |
| `skillOrWorkflow` | `"Skill"` | лӘ…мӢңм Ғ invocationмқҙ н•„мҡ”н•ҳл©ҙ `"Workflow"` мӮ¬мҡ© |
| `frontmatter` | `null` | platformмқҙ metadataлҘј мҡ”кө¬н•ҳл©ҙ YAML object м¶”к°Җ |

н”Ңлһ«нҸјл§ҲлӢӨ кё°лҢҖн•ҳлҠ” file naming conventionкіј metadata formatмқҙ лӢӨлҰ…лӢҲлӢӨ.
