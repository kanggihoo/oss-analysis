# oss-analysis LLM Wiki

This directory is the LLM-managed, long-term knowledge base for `/Users/kkh/Desktop/oss-analysis`.

## Start here

1. Read `SCHEMA.md` for rules, taxonomy, and evidence policy.
2. Read `index.md` to locate pages.
3. Check recent entries in `log.md` before editing.
4. Update pages under `projects/`, `concepts/`, `comparisons/`, or `queries/`.
5. Update `index.md` and append to `log.md` after every edit batch.

## Important boundary

The wiki does not replace the other workspace directories:

- `repos/` = source checkouts and primary evidence.
- `artifacts/` = analyzer output and external baselines.
- `reports/` = human-facing per-repo reports.
- `wiki/` = durable synthesis, cross-links, and reusable knowledge.
