# raw/

This directory is reserved for immutable source captures that do not already have a stable canonical location in the workspace.

Current policy:

- Do not duplicate full `repos/` checkouts into `wiki/raw/`.
- Do not duplicate full `artifacts/` analyzer outputs into `wiki/raw/`.
- Use `sources:` frontmatter to cite stable workspace paths such as `AGENTS.md`, `OSS_ANALYSIS_WORKFLOW.md`, `repos/<repo>/...`, and `artifacts/<repo>/...`.
- Add raw captures here only for small external articles, pasted notes, PDFs, transcripts, or source material that would otherwise be lost.

Subdirectories:

- `articles/`
- `papers/`
- `transcripts/`
- `assets/`
- `workspace/`
